
The AI’s job is not to obey it.
The AI’s job is to process it.

How the loop should work

The harness loop for conflict should be:
1. Collect
   Read all relevant project files and research notes.

2. Classify
   Mark each item as decision, option, assumption, risk, evidence, or unresolved.

3. Compare
   Find contradictions, overlaps, trade-offs, and hidden dependencies.

4. Resolve
   Decide whether the conflict is safe to resolve, needs architecture update, or needs Shaun.

5. Record
   Update DECISIONS-TO-MAKE, ARCHITECTURE, TECH-SPEC, DATABASE, TASKS, or AI-NOTES.

6. Continue
   Only move forward once the conflict has a status.
